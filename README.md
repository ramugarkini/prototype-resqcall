# ResQCall 🚨

**ResQCall** is a Flutter-based emergency assistance application designed to help users quickly send alerts and location details during critical situations. The app focuses on fast communication, real-time location sharing, and reliable emergency notifications.

---

## 📱 Features

* 🚨 **Emergency Alert**

  * Instantly send distress alerts to predefined emergency contacts.

* 📍 **Live Location Sharing**

  * Automatically share the user's current GPS location during emergencies.

* 📞 **Quick Call Support**

  * Directly call emergency contacts or services from within the app.

* 🔔 **Push Notifications**

  * Notify registered contacts when an emergency alert is triggered.

* 👤 **User Profile Management**

  * Manage personal details and emergency contact list.

* 🔐 **Secure Data Handling**

  * User data and alerts are securely processed.

---

## 🛠 Tech Stack

* **Frontend:** Flutter (Dart)
* **Platform:** Android / iOS
* **Location Services:** GPS / Geolocation APIs
* **Notifications:** Firebase Cloud Messaging (optional)

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/ramugarkini/prototype-resqcall.git
cd prototype-resqcall
```

### 2️⃣ Install Dependencies

```bash
flutter pub get
```

### 3️⃣ Run the Application

```bash
flutter run
```
